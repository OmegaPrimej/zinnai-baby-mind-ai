# 🧸 **PHASE 4 — PROTO‑MOTIVATION (REWARD SYSTEM)**

We add a simple but powerful reward channel that:

- Learns from **sensory surprise** (novelty = reward)
- Avoids **repetitive pain** (stuck states = punishment)
- Creates **approach / avoid** behavior

## 📁 New file: `zinnai/reward.py`

```python
import numpy as np

class RewardSystem:
    """
    Proto-motivation: rewards novelty, punishes stagnation.
    """

    def __init__(self, decay=0.95):
        self.last_state = None
        self.stuck_counter = 0
        self.decay = decay

    def compute(self, current_state, curiosity_drive):
        """
        Returns reward signal between -1 and 1.
        """
        if self.last_state is None:
            self.last_state = current_state
            return 0.0

        # Reward 1: change from previous state (novelty)
        delta = abs(current_state - self.last_state)
        novelty_reward = min(delta * 2, 1.0)  # scale to [0,1]

        # Reward 2: alignment with curiosity drive
        curiosity_alignment = 1.0 - abs(curiosity_drive - delta)

        # Punishment: being stuck (low delta for many steps)
        if delta < 0.01:
            self.stuck_counter += 1
        else:
            self.stuck_counter = max(0, self.stuck_counter - 1)

        stagnation_penalty = min(self.stuck_counter / 10.0, 0.5)

        # Combine
        reward = (novelty_reward * 0.6 + curiosity_alignment * 0.4) - stagnation_penalty
        reward = np.clip(reward, -1.0, 1.0)

        # Update
        self.last_state = current_state * self.decay + current_state * (1 - self.decay)
        return reward
```

Inside `__init__`:


Now the baby mind:

- **Loves novelty** → seeks new baby steps
- **Hates being stuck** → avoids repetitive loops
- **Grows faster when curious** → reinforcement learning at the infant level
